# Project: I2C OLED Eyes

In this project, I use different animations with the help of an OLED screen, using simple animation logic such as:

- Look left and right  
- Blink and fast blink  

This project demonstrates a basic use of animation with the help of an OLED display.

---

## List of Components

- 1 × ESP32 WROOM-32D (38 pin)
- 1 × OLED screen 0.96" (I2C)
- 4 × Jumper wires
