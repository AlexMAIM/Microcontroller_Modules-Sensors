# Project: LED – Button – Potentiometer

In this project, I control two LEDs using different input methods:

- The **blue LED** is turned ON and OFF using a **push button**.
- The **yellow LED** has adjustable brightness using a **5 kΩ potentiometer**.

This project demonstrates basic use of digital input, analog input, and PWM output with an Arduino UNO.

## List of Components
- 1× Arduino UNO R3  
- 2× 220 Ω Resistors  
- 2× LEDs (Blue & Yellow)  
- 1× Push Button  
- 1× 5000 Ω Potentiometer  
- 10× Wires (6 Dupont + 4 Jumper)
